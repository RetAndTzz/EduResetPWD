# EduResetPWD
Some tips help you to reset your edumail password
[1]域名:sunyit.edu
方式:学校官网更改
Url:http://banner.sunypoly.edu
过程:点击 Enter Secure Area
输入用户名 @前面的那串+密码 登录
点击Personal Information
然后点击Change your Banner PIN and Computer Account Password即可
